# ***ReadME***


Create a virtual environment
```shell
$ virtualenv .
```
Install requirements
```shell
$ pip install -r requirements.txt
```
Download spaCy en_core_web_sm model
```shell
$ python -m spacy download en_core_web_sm
```
Check spacy compatibility
```shell
$ python -m spacy validate
```
Run app
```python
$ python main.py
```
